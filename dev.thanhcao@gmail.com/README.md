# TanStack Query Learning Path

### Vấn đề khi không sử dụng Tanstack Query

**useEffect** chỉ là công cụ để kéo dữ liệu về chứ không thể quản lý dữ liệu đó ⇒ Thiếu bộ não xử lý trung tâm.&#x20;

**Gặp phải 3 trường hợp như sau:**

* Không biết khi nào data đã cũ để tự động gọi lại.
* Không biết các component khác cũng đang cần chính dữ liệu đó để dùng chung.
* Không biết dữ liệu trên server đã thay đổi để tự động cập nhật.
